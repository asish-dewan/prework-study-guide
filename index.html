Git

  • git clone: creates a working copy of a local repository
  • git status: checks what branch we are currently on
  • git checkout -b 'branch name': creates a new branch and switches to it
  • git add -A: adds all the modified files to the branch
  • git commit: Commit any files added with git add, and also commit any files changed since then
  • git pull origin main/master: Fetches the lastest version of the main branch from the server
  • git push origin main/master: Send the changes to the main/master branch of your remote repository
